Description of API's in Application

API Name Request Type API endpoint Description Authorization Required
Login POST /auth/login/ To authenticate a user and provide the access token No
Logout GET /auth/logout/ To revoke an access token and log out of the session Yes
Register POST /auth/register/ To register an account in Application using E-mail ID and Name and user will get credentials delivered in their Email-ID No
Books And Authors POST /add/book/author/ To add one or more books and authors and create a relation between them. Yes
Books and Category POST /add/category/book/ To add one or more category create a relation between category and books. Yes
Books and Author Details GET /book/detials/ To fetch details of books along with the authors detials. Yes
Author under Category GET /api/analytics/author/:categories_id To fetch the distinct authors count under a category id provided in request. Yes
Books under Category GET /api/analytics/books/:categories_id To fetch the distinct books count under a category id provided in request. Yes

*Note for request body kindly visit the API endpoint. For every request the body has been suggested.
